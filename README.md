# Tourism2Trip 🌍

Tourism2Trip is a travel and tourism website designed to showcase destinations, travel packages, and booking information in a clean and user-friendly interface.

The project focuses on responsive design, structured layout, and interactive user experience.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The website will open in your browser.

No additional setup required.
